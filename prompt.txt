built-in tooltip_text property. Use the existing TEST_CARD_DATA "desc"
field as the tooltip text. Smallest safe patch only.

In _ready(), after the starter button connections, add a loop that
sets tooltip_text on every DevDock button:

    # Set tooltips on dev card buttons
    for container in _button_containers:
        for child in container.get_children():
            if child is Button and TEST_CARD_DATA.has(child.text):
                child.tooltip_text = TEST_CARD_DATA[child.text]["desc"]

description. No separate tooltip dictionary needed — TEST_CARD_DATA
already has the data.
