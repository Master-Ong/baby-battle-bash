## Coordinate System Rules

### CanvasLayer (screen space)
- Origin is TOP-LEFT corner of screen (0, 0)
- Used for: HP label, Energy label, Mob name/HP labels, Deck label, Graveyard label, End Turn button
- Positions use offset_left and offset_top

### Node2D (world space)
- Origin is CENTER of screen (0, 0)
- Left edge X=-576, Right edge X=576, Top Y=-324, Bottom Y=324
- Used for: HandZone_Area2D, PlayerField_Node2D, CardSlots, MobContainerNode2D, DG_Node2D, all panels

### During drag and drop
- Always use get_global_mouse_position()
- Always use global_position when moving a dragged card
- For slot detection use Rect2.has_point() on SlotPanel.get_global_rect()
- Use _input_event for press detection, _unhandled_input for release detection

## Card Type Rules

| Type | On Click/Drop | Energy Cost | Destination |
|------|--------------|-------------|-------------|
| ANIMAL | Drag to field slot | 0 — FREE | Stays on field until defeated |
| ATTACK | Single click | 1 | Deals damage to mob, goes to graveyard |
| SKILL | Single click | 1 | Gives player defense, goes to graveyard |
| COLOR | Single click | 1 | Special effect, goes to graveyard |
| WORD | Single click | 1 | Special effect, goes to graveyard |

## Combat Flow Rules

Turn order every End Turn:
1. Each field animal attacks mob
2. Mob attacks each field animal — mob_damage = 10
3. Dead animals removed from field_slots, sent to graveyard
4. If field is empty after mob attacks — take_player_damage(mob_damage)
5. spawn_field_visuals() to refresh display
6. discard_hand() — all hand cards go to graveyard
7. draw_cards(5) — draw new hand

Energy resets to max_energy at start of each turn.
Defense resets to 0 at start of enemy turn.

---

## Field Slot Rules

- field_slots is a Dictionary {0: null, 1: null, 2: null}
- Slot 0 = left (CardSlot1_Area2D)
- Slot 1 = center (CardSlot2_Area2D)
- Slot 2 = right (CardSlot3_Area2D)
- Never use field_animals array — it no longer exists

## Critical Rules — Never Break These

1. CombatScene root Node2D position must always be Vector2(0,0)
2. Cards use Area2D not CharacterBody2D
3. MobContainerNode2D is the ONLY node swapped between encounters
4. Use create_tween() for all animations — never the old Tween node
5. Hand cap is 10 cards maximum
6. Graveyard shuffles back into deck when deck runs empty
7. No shirk in game lore — no false gods, idol worship, or polytheistic themes
8. field_animals array does not exist — use field_slots Dictionary
9. queue_free() for hand visuals, free() for field visuals during spawn_field_visuals()
10. Always erase card from hand before calling draw_cards() to avoid duplicates
