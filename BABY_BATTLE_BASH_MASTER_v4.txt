## CURRENT STATE — WHAT WORKS ✅

**Core Loop:**
- Full roguelike loop: Main Menu → Animal Select → Roadmap → Combat → Reward → Roadmap
- Lose screen with New Run and Main Menu
- Run Complete screen (after boss) with clean Main Menu return
- GameState.reset_run() works cleanly between runs

**Map 1 Content:**
- 7 roadmap nodes: Combat, Rest, Combat, Combat, Shop, Chest, Boss
- Rest Site heals 15 HP
- Chest Site gives random relic from pool
- Boss node: Baby Dragon (90 HP, 16 ATK)

**Gold System:**
- GameState.gold starts at 0
- Normal combat reward awards +10 gold
- Gold accumulates during run
- Gold resets on new run
- Boss victory does not award gold

**Relic System:**
- GameState.relics is an Array of strings
- Relic pool in chest_site.gd with 3 relics:
  - Bandage Roll: 5 Block on turn 1
  - Iron Shield: 3 Block on turn 1
  - Lucky Coin: +5 gold after normal combat
- Chest randomly picks one unowned relic
- Duplicate prevention (won't offer what you already own)
- "No new relics" when all 3 owned
- Relics stack (Bandage Roll + Iron Shield = 8 Block turn 1)
- DevDock relic toggle row on Animal Select (3 buttons)

**Shop System (3-Tab):**
- Tab 1 — Market: 3 unique card offers, Adopt (10g), Reroll (1g per slot)
- Tab 2 — Training Camp: Fusion v0 (2 Bunny→Bunny+, 2 Turtle→Turtle+, 2 Dog→Dog+)
- Tab 3 — Card Removal: Remove earned cards for 3g, grouped "Village Market Stalls" UI
- Card Removal has 4 stall columns (Animals, Colors, Adjectives, Verbs/Skills)
- Master confirmation box at bottom (select then confirm)
- Starter deck cards don't appear in removal (they're not in reward_cards)
- Return to Roadmap preserves run state

**Fusion/Evolution v0:**
- Uses exact card_name matching (Bunny+ does NOT count as Bunny)
- Removes 2 base cards, adds 1 upgraded card from CardDatabase
- Free (no gold cost)
- Only works on reward/shop/test-selector animals (NOT starter deck copies)
- STATUS: Tested and working

**Shared CardDatabase:**
- res://Scripts/CardDatabase.gd
- Static functions: get_animals(), get_colors(), get_adjectives(), get_attacks(), get_skills(), get_upgraded_animals()
- Rewards and Shop both use shared definitions

**Card Pool:**
- Animals: Bunny (4/12), Turtle (2/20), Dog (8/10)
- Upgraded: Bunny+ (6/16), Turtle+ (4/28), Dog+ (12/14)
- Colors: Blue (draw 1), Red (+2 ATK), Green (heal 3), Yellow (gain 1 energy), Purple (6 Block), White (+2 ATK/+4 HP to field animals)
- Adjectives: BIG, FAST, TOUGH, SHARP, SMART, MAGIC
- Attacks: Strike, Scratch, Bite, Dash (4 dmg + draw 1), Pounce, Roar
- Skills: Defend, Hide, Run Away, Block, Curl Up, Guard, Scout (1 block + draw 1), Sneak (4 block + draw 1)

**Mob Pool:**
- Early: Mr. Kiwi (40/8), Prickly Hedgehog (35/10)
- Mid: Big Bear (80/12), Swift Fox (50/10), Sleepy Owl (55/9), Muddy Piglet (45/11)
- Hard: Stone Turtle (90/14), Sneaky Raccoon (70/15)
- Boss: Baby Dragon (90/16)
- Tier-based randomization, no immediate repeats

**Animal Select DevDock (Test Tools):**
- 5 rows: Animals, Colors, Adjectives, Verbs, Relics
- Toggle buttons with tooltips
- Selected cards/relics append to starter deck/GameState.relics

**Combat Systems:**
- Combat end-state handling is stable (combat_ended guard, deferred outcome check)
- Block system works (player_defense consumed before HP)
- Field animals with 0 HP are removed to graveyard
- Negative HP clamped to 0 for display
- field_slots is the source of truth
- Card HP mutation bug FIXED (cards cloned when building combat deck)
- All color card handlers are explicit (Blue, Red, Green, Yellow, Purple, White)
